# DigiSurf Australia — AI Automation Agency

**Sydney-based AI automation for Australian businesses.**

Custom AI solutions for healthcare, fitness, real estate, retail, and content creators.

🌐 [digisurfaustralia.com.au](https://digisurfaustralia.com.au)

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Services

- AI Chatbots (website, WhatsApp, Messenger)
- Content Automation (clip, caption, schedule)
- Document & Data Automation
- Business Process Automation (healthcare, fitness, real estate)
